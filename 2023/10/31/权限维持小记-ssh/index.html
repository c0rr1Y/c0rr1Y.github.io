如果你想通过iptables添加规则允许任何IP连接SSH，你可以使用以下命令。这里假设你使用的是默认的SSH端口22。

iptables -A INPUT -p tcp –dport 10086 -j ACCEPT

这个命令添加一个规则，允许TCP流量通过到达目标端口22（SSH端口）。 -A INPUT 意味着将规则添加到输入链（incoming chain）。
